# Digital Identity Protection System - Installation & Setup Guide\n\n## Prerequisites\n\n- Node.js v16+ (or v18+ recommended)\n- npm or yarn package manager\n- PostgreSQL 13+\n- Docker & Docker Compose (optional, for containerized setup)\n- Git\n\n## Quick Start with Docker\n\n### 1. Clone Repository\n```bash\ngit clone https://github.com/bonitavivian06-hub/digital-identity-protection-.git\ncd digital-identity-protection-\n```\n\n### 2. Setup Environment Variables\n```bash\ncp .env.example .env\n```\n\nEdit `.env` and configure:\n```env\nNODE_ENV=development\nPORT=5000\nDB_HOST=postgres\nDB_USER=postgres\nDB_PASSWORD=your_secure_password\nDB_NAME=digital_identity_protection\nJWT_SECRET=your_very_long_random_secret_key_here\nENCRYPTION_KEY=your_encryption_key_here\n```\n\n### 3. Start Services\n```bash\ndocker-compose up --build\n```\n\nServices will be available at:\n- Frontend: http://localhost:3000\n- Backend API: http://localhost:5000\n- PostgreSQL: localhost:5432\n- Redis: localhost:6379\n\n## Manual Setup (Without Docker)\n\n### Backend Setup\n\n```bash\n# Navigate to backend directory\ncd backend\n\n# Install dependencies\nnpm install\n\n# Create .env file\ncp ../.env.example .env\n\n# Configure environment variables\n# Edit .env with your database credentials\n\n# Run database migrations\nnpm run migrate\n\n# Start development server\nnpm run dev\n```\n\nBackend runs on `http://localhost:5000`\n\n### Frontend Setup\n\n```bash\n# Navigate to frontend directory\ncd frontend\n\n# Install dependencies\nnpm install\n\n# Configure API URL\n# Update REACT_APP_API_URL in .env if needed\n\n# Start development server\nnpm start\n```\n\nFrontend runs on `http://localhost:3000`\n\n### Database Setup\n\n```bash\n# Create database\ncreatedb digital_identity_protection\n\n# Apply initial schema\npsql digital_identity_protection < database/init.sql\n\n# Or use npm script\nnpm run db:migrate\n```\n\n## Project Database\nDB_HOST=localhost\nDB_PORT=5432\nDB_NAME=digital_identity_protection\nDB_USER=postgres\nDB_PASSWORD=postgres_password\n\n# Authentication\nJWT_SECRET=your_jwt_secret_here_min_32_chars\nJWT_EXPIRY=24h\nREFRESH_TOKEN_SECRET=your_refresh_token_secret_here\n\n# Encryption\nENCRYPTION_KEY=your_encryption_key_here_32_chars\n\n# Redis\nREDIS_HOST=localhost\nREDIS_PORT=6379\n\n# API\nREACT_APP_API_URL=http://localhost:5000\nCORS_ORIGIN=http://localhost:3000\n\n# Third-party APIs\nBREACH_API_KEY=your_breach_database_api_key\nEMAIL_SERVICE_API_KEY=your_email_service_api_key\nSMS_SERVICE_API_KEY=your_sms_service_api_key\n\n# Security\nRATE_LIMIT_WINDOW=15m\nRATE_LIMIT_MAX_REQUESTS=100\n\n# Logging\nLOG_LEVEL=info\nLOG_FILE=./logs/app.log\n```\n\n## Database Connection\n\n### PostgreSQL Connection String\n```\npostgresql://username:password@localhost:5432/digital_identity_protection\n```\n\n### Connection Pool Settings\n- Max connections: 20\n- Idle timeout: 30 seconds\n- Connection timeout: 2 seconds\n\n## API Testing\n\n### Using curl\n```bash\n# Health check\ncurl http://localhost:5000/health\n\n# Register user\ncurl -X POST http://localhost:5000/api/auth/register \\\n  -H }'\n\n# Login\ncurl -X POST http://localhost:5000/api/auth/login \\\n  -H \"Content-Type: application/json\" \\\n  -d '{\n    \"email\": \"user@example.com\",\n    \"password\": Error\n```bash\n# Check PostgreSQL is running\npsql --version\n\n# Test connection\npsql -h localhost -U postgres -d digital_identity_protection\n\n# View connection logs\ndocker-compose logs postgres\n```\n\n### Redis Connection Error\n```bash\n# Check Redis is running\nredis-cli ping\n\n# View Redis logs\ndocker-compose logs redis\n```\n\n### Module Not Found\n```bash\n# Reinstall dependencies\nrm -rf node_modules package-lock.json\nnpm install\n```\n\n### Build Errors\n```bash\n# Clear cache and rebuild\nnpm run clean\nnpm install\nnpm run build\n```\n\n## Production digital_identity_protection > backup.sql\n\n# Restore\npsql digital_identity_protection < backup.sql\n```\n\n### Docker Volume Backup\n```bash\n# Backup PostgreSQL volume\ndocker run --rm -v postgres_data:/data -v $(pwd):/backup \\\n  alpine tar czf /backup/postgres_backup.tar.gz -C /data .\n\n# Restore\ndocker run --rm -v postgres_data:/data -v $(pwd):/backup \\\n  alpine tar xzf /backup/postgres_backup.tar.gz -C /data\n```\n\n## Support & Resources\n\n- **Documentation**: See README.md and SYSTEM_ARCHITECTURE.md\n- **Issues**: Create an issue on GitHub\n- **Email**:
